# Clion Key

| 快捷键                         | 功能                                   |
|-----------------------------|--------------------------------------| 
| Ctrl + 空格 -> Enter [OR] Tab | 第一步查看补全建议 -> Enter 在前方插入 [OR] Tab 替换 |
| Ctrl + B                    | 前往声明                                 |
| Ctrl + D                    | 复制当前行并复制                             |
| Ctrl + E                    | 查看最近的文件                              |
| Ctrl + N                    | 搜索类                                  |
| Ctrl + R                    | 查替换                                  |
| Ctrl + G                    | 跳转到指定行                               |
| Ctrl + W                    | 可选择文本光标处的单词                          |
| Ctrl + Q                    | 预览文档                                 |
| Ctrl + ;                    | AceJump 快速跳转                         |
| Ctrl + Alt + ;              | AceJump 快速跳转并选中单词                    |
| Ctrl + Shift + ;            | AceJump 快速跳转并选中单词                    |
| Ctrl + Alt + Shift + N      | 查找方法或全局变量                            |
| Ctrl + Shift + N            | 查找文件                                 |
| Ctrl + F                    | 打_在文件中查找                             |
| Ctrl + Shift + 逗号           | 在文件中查找                               |
| Ctrl + Shift + R            | 在文件中替换                               |
| Ctrl + F12                  | 打开文件结构                               |
| Alt + 7                     | 在工具栏中打开文件结构                          |
| Alt + Enter                 | 显示上下文操作                              |
| ---                         | ---                                  |
| Tab + C                     | 转移到一个非白字符                            |
| Tab + D                     | # -- 快捷输入 `#`                        |
