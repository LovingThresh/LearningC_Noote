# QString

| 成员函数              | 作用说明      |
|-------------------|-----------|
| isEmpty()         | 判断是否为空    |
| unicode()         | UTF-16编码值 |
| isDigit()         | 是否为数字0~9  |
| isLetter()        | 是否为字母     |
| isLetterOrDigit() | 是否为数字或字母  |
| isUpper()         | 是否为大写     |
| isLower()         | 是否为小写     |
| isMark()          | 是否为符号     |
| isSpace()         | 是否为空格     |
| isSymbol()        | 是否为符号     |
| isPunct()         | 是否为标点符号   |
