# About How to Repetition

## 复现流程 
1. 找好文章！然后阅读论文并理解文章意图，确定复现的目的
   1. 仔细阅读原论文，理解作者的提出的问题、采用的方法，以及实验的设计和结果
   2. 思考该论文的结果对于当前研究的价值，是数据处理的方式、模型的构建、训练流程的创新、损失函数的设计或结果后处理的优化？
   需要清楚的是复现的目的，而不能为了复现而复现（除非处于研究的初期）。
   3. 思定而后动的好处是将精力花在复现关键点（组件）上，而不是复现论文上。 复现整个项目可能花费的资源与精力很有可能与实际收获不符，而复现组件具有更高的性价比。

2. 环境配置问题
    - 组内目前提供了四种环境配置以供选择
        1.  22_PT: PyTorch 1.13 版本，按理来说应该是非常稳定的版本，绝大部分的第三方库与项目都能基于该镜像进行研究
        2.  04_PT: PyTorch 2.0 版本，PyTorch 更新后的稳定版本，具有compile的特性。之前大部分的项目都是基于该镜像，内置丰富的第三方库
        3.  23_PT: PyTorch 2.2 版本，23年12月最新的版本，拥有新特性，可以尝试FP8的训练，但是目前很多第三方库还没有提供相应的原生支持，有些第三方库可能需要从源码进行构建，如果没有信心请使用之前的版本。
        4.  TF   : TensorFlow 2.7 版本，可在遇到TensorFlow项目时使用，但尽可能避免TF项目，不推荐使用TensorFlow进行开发。对RepairerGAN项目感兴趣可以使用该镜像。
    - 第三方库安装
        1. 这些镜像已经内置了很多第三方库，一般来说，绝大部分项目只需要使用 pip 补充一下，很多项目已经告知了requirements.txt，非常方便。
        2. 使用IDE打开项目，多打开几个python文件，如果缺少某个库文件的话，IDE一般会使用红线标识。
        3. `pip install [library] -i https://pypi.tuna.tsinghua.edu.cn/simple`
        4. 如果需要指定版本的请使用 `pip install [library]==[version] -i https://pypi.tuna.tsinghua.edu.cn/simple`
    - 从源码安装第三方库
        1. 如果使用pip无法搜索到某些第三方库，亦或者安装失败，则可以尝试从源码进行安装。
        2. 安装的方式是 从 Github 上获取源代码，可以使用 git clone（可能失败），也可以直接下载压缩包进行解压到目录中。
        3. 打开终端或命令行，进入解压的源码目录中，检查和满足依赖项，请先安装好需要的前置库。
        4. 执行安装命令 `python setup.py install` ，等待系统安装完成。
        5. 测试安装，在 Python 解释器中尝试导入库，`import [library]`
    - 想要练手的话，可以尝试一下[RepViT](https://github.com/THU-MIG/RepViT)，看能不能跑出测试结果。
